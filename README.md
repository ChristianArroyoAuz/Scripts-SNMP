# Scripts-SNMP
 Scripts de SNMP que muestra los hosts activos, puertos activos de TCP y UDP. Adicionalmente puede realizar peticiones SNMPv1.
 Se muestra las caracteristicas de las MIBS del host activo seleccionado.
 Se debe ingresar una direccion ip y ua mascara. Ejemplo: 192.168.200.1 255.255.255.0
 Se debe ingresar el OID de la MIB. Ejemplo: 1.3.6.1.2.1.1.5.0 ó sysName.o
